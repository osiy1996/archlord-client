========================================================================
    MFC 라이브러리 : AgcmDialogs 프로젝트 개요
========================================================================


응용 프로그램 마법사에서 이 AgcmDialogs DLL을 만들었습니다.
 이 DLL은 MFC의 기본적인 사용법을 보여 줄 뿐아니라
사용자가 DLL을 만드는 데 있어 출발점이 될 수 있습니다.

이 파일에는 AgcmDialogs 응용 프로그램 DLL을 구성하는 각각의 파일에
들어 있는 요약 설명이 포함되어 있습니다.

AgcmDialogs.vcproj
    응용 프로그램 마법사를 사용하여 생성한 VC++ 프로젝트의 기본 프로젝트 파일입니다. 
    해당 파일을 생성한 Visual C++의 버전 정보를 비롯하여 
    응용 프로그램 마법사에서 선택한 플랫폼, 구성 및 
    프로젝트 기능에 대한 정보가 들어 있습니다.

AgcmDialogs.cpp
    DllMain()이 정의되어 있는
    기본 DLL 소스 파일입니다.
AgcmDialogs.rc
    프로그램에서 사용하는 모든
    Microsoft Windows 리소스의 목록입니다. RES 하위 디렉터리에 저장된
    아이콘, 비트맵, 및 커서 등이 여기에 포함됩니다. 이 파일은
    Microsoft Visual C++에서 직접 편집할 수 있습니다.

res\AgcmDialogs.rc2
    이 파일에는 Microsoft Visual C++로 편집할 수 없는 
    리소스가 들어 있습니다. 리소스 편집기로 편집할 수 없는
    리소스는 모두 이 파일 안에 배치해야 합니다.

AgcmDialogs.def
    이 파일에는 DLL에 대한 정보가 들어 있으며
    이 정보는 Microsoft Windows에서 실행되기 위해 반드시 필요합니다.
    이 파일에서는 DLL의 이름 및 설명 같은 매개 변수를 정의하며,
    해당 DLL에서 함수를 내보내기도 합니다.

/////////////////////////////////////////////////////////////////////////////
기타 표준 파일:

StdAfx.h 및 StdAfx.cpp는
    AgcmDialogs.pch라는 이름의 PCH(미리 컴파일된 헤더) 파일과
    StdAfx.obj라는 이름의 미리 컴파일된 형식 파일을 빌드하는 데 사용됩니다.

Resource.h
    새 리소스 ID를 정의하는 표준 헤더 파일입니다.
    Microsoft Visual C++에서 이 파일을 읽고 업데이트합니다.
